# PassHub

> 开源 otp 动态口令

### 介绍

OTP (One Time Programmable) 双因素认证

具体信息大家自己百度

**要实现的对接**

开放了接口，集成即可。
- Linux ssh （已经实现）
- Windows  3389  （已经实现）
- WordPress 登录
- 等等 。。。

**平台特色功能**

- 口令共享  可以多人使用一个 口令
- 权限控制 可以控制某些人 使用哪些口令
- 任意创建 创建属于每台设备的专用口令
